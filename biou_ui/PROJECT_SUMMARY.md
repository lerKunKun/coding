# BIOU 前端项目总结

## 项目概述

本项目是一个基于 React + TypeScript + Ant Design 的现代化后台管理系统，严格遵循字节跳动前端开发规范。项目采用组件化开发模式，具有良好的代码组织结构和可维护性。

## 技术特色

### 🎨 设计规范
- **字节跳动设计语言**: 采用字节跳动官方设计规范
- **主题色彩**: #1664FF 主色调，配色协调统一
- **组件拆分**: 严格控制组件代码行数，提高可维护性

## 部署说明

### 开发环境
```bash
npm install
npm start
```

### 生产环境
```bash
npm run build
# 将 build 目录部署到 Web 服务器
```

### 环境配置
